Git Setup

Set up a GitHub account

  1. Go to github.com and register for a free account.
  2. In your account, click on your picture in the upper-right corner, and select Settings.
  3. Select Emails from the list of personal settings in the right-hand-side menu.
  4. Check both of these options, so they are selected: Keep my email address private and Block command line pushes that expose my email.
  5. Under the Keep my email address private option note the email address alias that you will use later when setting up your Git account on your local machine. My alias is, for example, p-buda@users.noreply.github.com.
    Why do we do this? Since GitHub free repositories are public, your email address would be exposed whenever you commit code to those public repositories. Using alias hides your actual email address.

Now you are ready to set up Git application on your personal computer.

Set up Git on your local machine

  1. Download Git and install it like any other, standard application. Note for the Mac users: In order to open the installer, after you open the downloaded package, you might need to right-click on the installer and then click Open.
  2. During the installation process, you can leave all the default options as they are.
  3. Open Git Bash application if you have Windows machine or open Terminal if you have a Mac/Linux.
    Note that Git is a tool that works inside the Terminal and whether you are on a Mac or Windows, you can use all the standard commands to create folders (mkdir), move in the directory structure (cd), list files (ls), and similar.
  4. Set up your global settings for Git, like so:
    • In the Terminal/Git Bash type git config --global user.name "First Last". Replace the "First Last" with your actual first and last name or a nickname. Hit Enter to save this setting. This will be publicly visible once you connect your Git to GitHub. I do use my actual first and last name though.
    • Still in the Terminal/Git Bash, type git config --global user.email "email@example.com". Replace the "email@example.com" with the alias email you got from setting up your GitHub account above. Mine was p-buda@users.noreply.github.com.
    You set up the global settings only once and all the projects that you track with Git will have that information. If you would like to change your name or email, just retype the above command(s) again with the new settings.
